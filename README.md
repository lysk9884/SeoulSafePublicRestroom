# 주요 기능
1. 인근 화장실 검색  
GPS 기능을 사용하여 근방에 있는 화장실을 찾아줍니다.  
마커를 클릭하면 점검 내용, 최근 점검 일자, 화장실 안전도 정보를 확인할 수 있습니다.

2. 안심보안관의 점검 내용 기록  
안심보안관은 NFC 태그를 이용해 화장실 안전 등급을 설정하거나 점검 내용을 기록할 수 있습니다.

3. 몰카 의심 제보
몰카가 의심스러운 현장을 글과 사진을 첨부해 제보할 수 있습니다.
